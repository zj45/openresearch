Execute a bash command on a remote server via SSH.

- Connects either with direct SSH credentials or via a host defined in an SSH config file
- Runs the given command on the remote server and returns stdout + stderr
- The command runs in the remote user's default shell
- Default timeout is 120 seconds; override with the timeout parameter
- Use this tool when you need to run commands on a remote machine (e.g., experiment servers, GPU nodes)
