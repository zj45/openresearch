Convert a PDF file to Markdown text.

Usage:
- The filePath parameter should be an absolute path to the PDF file.
- Returns the PDF content converted to Markdown format.
- IMPORTANT: Always try the read tool first to read PDF files. Only use this convert tool as a fallback when the read tool fails or returns unreadable content for a PDF.
- If you are unsure of the correct file path, use the glob tool to look up filenames by glob pattern.
